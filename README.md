# vlingo-symbio-jdbc

[![Javadocs](http://javadoc.io/badge/io.vlingo/vlingo-symbio-jdbc.svg?color=brightgreen)](http://javadoc.io/doc/io.vlingo/vlingo-symbio-jdbc) [![Build](https://github.com/vlingo/vlingo-symbio-jdbc/workflows/Build/badge.svg)](https://github.com/vlingo/vlingo-symbio-jdbc/actions?query=workflow%3ABuild) [![Download](https://img.shields.io/maven-central/v/io.vlingo/vlingo-symbio-jdbc?label=maven)](https://search.maven.org/artifact/io.vlingo/vlingo-symbio-jdbc) [![Gitter chat](https://badges.gitter.im/gitterHQ/gitter.png)](https://gitter.im/vlingo-platform-java/symbio)

The VLINGO XOOM SYMBIO implementation for JDBC for Reactive storage using Event-Sourcing, Key-Value, and Object storage.

Docs: https://docs.vlingo.io/vlingo-symbio

### Installation

```xml
  <dependencies>
    <dependency>
      <groupId>io.vlingo</groupId>
      <artifactId>vlingo-symbio</artifactId>
      <version>1.6.0</version>
      <scope>compile</scope>
    </dependency>
    <dependency>
      <groupId>io.vlingo</groupId>
      <artifactId>vlingo-symbio-jdbc</artifactId>
      <version>1.6.0</version>
      <scope>compile</scope>
    </dependency>
  </dependencies>
```

```gradle
dependencies {
    compile 'io.vlingo:vlingo-symbio:1.6.0'
    compile 'io.vlingo:vlingo-symbio-jdbc:1.6.0'
}
```
